NAME

adt_binarytree - C functions for binary trees

SYNOPSIS

#include "binaryTree.h"

tree *
tree_create(int number, void * label, tree * father, tree * left, tree * right);

void
tree_destroy(tree ** t);

void
tree_updateFathers(tree * t);

void
tree_print(tree * t, void (labelPrint)(void ));

int
tree_height(tree * t);

tree *
tree_numberSearch(tree * t, int number);

bool
tree_exist(tree * t);

bool
tree_isLeaf(tree * t);

bool
tree_isFather(tree * t);

DESCRIPTION

The tree functions allows creation, manipulation, and printing of tree in C. Labels inside the tree are void *, everything can be pointed behind the label.
The structure of a tree is defined in "binaryTree.h" as: 

typedef struct tree_s{
 int number;
 void * label;
 struct tree_s * left;
 struct tree_s * right;
 struct tree_s * father;
} *tree*;

If a tree does not have a left-tree, a right-tree or a father (the root), it needs to be set to NULL.

OPTIONS

bool tree_printDebug; -- If true, all datas of the tree structure will be displayed by tree_print
