NAME

adt_labels - C functions for labels

SYNOPSIS

#include "labels.h"

void
label_printInt(void * e);

void
label_printLong(void * e);

void
label_printDouble(void * e);

void
label_printBool(void * e);

void
label_printPointer(void * e);

void
label_printChar(void * e);

void
label_printString(void * e);

DESCRIPTION

The label_print* functions allows printing a label.
